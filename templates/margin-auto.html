Setting the width of an element will prevent it from stretching out to the edges of its container to the left and right. Then, you can set the left and right margins to auto to horizontally center that element within its container. The element will take up the width you specify, then the remaining space will be split evenly between the two margins.

This works with elements that are display: block;. This is what elements like <div> are by default. You can always set display: block; in CSS yourself.
