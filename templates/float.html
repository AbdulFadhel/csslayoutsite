Another CSS property used for layout is float. Float is intended for wrapping text around images, like this:

{% highlight css %} img { float: right; margin: 0 0 1em 1em; } {% endhighlight %}

Because there is no easy way to do layout in CSS, a lot of people use float for layouts like in the previous example. This can work, but I recommend avoiding it if you can stick to position. Since float wasn't made for complicated layouts, it can be unpredictable in complicated scenarios.
