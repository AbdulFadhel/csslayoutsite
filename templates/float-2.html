You can use multiple floated elements together to have them wrap when the page is resized.

{% highlight css %} .box { float: left; width: 200px; height: 100px; margin: 1em; } .after-boxes { clear: left; } {% endhighlight %}

{% for i in (1..11) %}

<div class="box">

I feel like I'm floating!

</div>
{% endfor %}
<div class="after-boxes">

You need to put an element with the clear property after your floated elements. Otherwise it will wrap around the floated elements, like in the previous page!

In this example I use a clear value of left because we are clearing elements that are floated to the left. You can also clear right and both.

</div>
